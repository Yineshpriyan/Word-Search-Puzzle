Word Search Puzzle
---------------

Requirements:
(word_search_grid.csv)
  If you have a image of the puzzle. You have convert it into a CSV File. (like Each letter in a cell)
(word list)
  Then you need to get the words to find out in the grid.
  make sure you arrange the words in one line in comma separated format.
  Eg: 
  absolute, adopt, aggressive, ashamed, blind, burn, cancel, civil, combination, declare, decrease, dismiss, examination, forgive, hire, impress, lower, measurement, patient, relief, schedule, scheme, tune, willing, wrong


Steps:
1. Run the index.html file.
2. Select the CSV file.
3. Enter the list of words (in comma separated format)
4. Hit search.


At the bottom you will see the words and it's starting and endiing positions.
There you go ...
